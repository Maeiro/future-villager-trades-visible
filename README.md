Future Villager Trades Visible
==============================

Forge (1.20.1) mod to display future villager trades in the trading screen.

Features
--------
- Shows future trades in the offer list without reopening the UI.
- Prevents interactions with trades that are not unlocked yet.
- Compatible with trade cycling (updates on cycle).

Requirements
------------
- Java 17

Dev run
-------
- `./gradlew runClient`
- `./gradlew runServer`
- `./gradlew runData`
